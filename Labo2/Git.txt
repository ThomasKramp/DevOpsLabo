Install Git on Ubuntu:
	sudo apt-get install git

Add Name and Email for Git-Logs:
	git config --global user.name ThomasKramp
	git config --global user.email s115472@ap.be

Make a Git Repo:
	git init

Check Repo Status:
	git status

Add File to Repo:
	git add <file / directory name>

Add All Files to Repo:
	git add .

Commit File:
	git commit -m "<discription of message>"

Show Git Log:
	git log

Checkout Commit:
	git checkout <commit-id>

Make Branch:
        git branch <branch-name>

Show Branches:
        git branch

Checkout Branch:
	git checkout <branch-name>

Merge External Branch into Current Branch:
	git merge <external-branch-name>

Using Github:
	git remote add <repo name> <Github Repo URL>
	git push -u <repo name> <branch>
	git pull -u <repo name> <branch>

Clone a Repo:
	git clone <Github Repo URL>
	(Doe dit nooit in een andere repo. Zorgt voor problemen)
